# 웹툰 데이터 큐레이션 가이드

## 목표
40~60개 작품 (HL 16~24개 + BL 24~36개)을 레진 공개 페이지에서 직접 수집.

## 작품당 수집 항목

| 필드 | 어디서 |
|---|---|
| `slug` | URL `lezhin.com/ko/comic/{slug}` 에서 추출 |
| `title` | 작품 페이지 헤더 |
| `author` | 작품 페이지 작가명 |
| `track` | 장르가 BL이면 "BL", 아니면 "HL" |
| `tags` | 작품 페이지 #해시태그 영역 (5개 이상) |
| `popularity` | 본인 판단 0~100 (인기 랭킹 / 별점 / 회차수 종합) |
| `synopsis` | 자체 작성 2~3문장 (공식 문구 복사 ❌) |
| `lezhinUrl` | `https://www.lezhin.com/ko/comic/{slug}` |
| `thumbnailUrl` | 우클릭 → 이미지 주소 복사 (보통 `ccdn.lezhin.com/v2/comics/.../tall.jpg?updated=...&width=840`) |
| `fallbackColor` | 작품 분위기에 맞는 hex (#FF6BCB/#D946EF/#7FFFD4/#FFB13B 등) |

## 균형 체크리스트
- [ ] 8개 결과 유형마다 최소 5개 이상 매칭 가능 (primaryTags 기준)
- [ ] 작가 1명당 최대 3개
- [ ] 19금/전연령 비율 ~ 4:6 (레진 실제 비율 반영)

## 결과 유형별 매칭 태그 참조
- 🌸 순정 로맨스 본진러 (HL): 순애, 사내연애, 다정남
- 👑 로판 여왕 (HL): 로판, 판타지, 결혼
- 🎭 다크 드라마 매니아 (HL): 드라마, 느와르, 미스터리
- ⚔️ 액션 판타지 헌터 (HL): 액션, 판타지, 능력남
- 💕 달달 BL 본진러 (BL): 달달물, 사랑꾼공
- 🔥 위험한 BL 헌터 (BL): 19금, 능욕공
- ⛓️ 집착 BL 마니아 (BL): 집착공, 까칠공
- 🐺 판타지 BL 덕후 (BL): 로판, 이종족
